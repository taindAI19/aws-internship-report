### In this section, you will learn how to deploy and invoke **AWS Agent Core** directly from your local machine.

### Why you should use **AWS CLI**

Using the AWS CLI allows you to easily access, configure, and manage Agent Core from your own environment.
It provides flexibility, convenience, and full control when working locally.
