### Week 1 Achievements:

* Understand what AWS is and grasp the basic service groups:
  * Compute
  * Storage
  * Networking
  * Database
  * Management Tools

* Successfully created and configured an AWS Free Tier account.

* Complete 5 tasks to receive an additional $100 credit

* Get familiar with AWS Management Console and know how to find, access, and use services from the web interface.

* Install and configure AWS CLI on your computer including:
  * Access Key
  * Secret Key
  * Default Region

* Use AWS CLI to perform basic operations such as:
  * Check account information & configuration
  * Get a list of regions
  * View EC2 services
  * Create and manage key pairs
  * Check running service information

* Be able to connect between the web interface and CLI to manage AWS resources in parallel.

* Know how to manage costs with AWS Bugets

* Understand how to access and create support requests on AWS Support
